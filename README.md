# Интеллектуальный ассистент для генерации деловой переписки

Данный проект призван решить проблему высокой длительности при формировании ответа на письмо.
Результатом разработки является интеллектуальная подсистема для сквозной автоматизации создания,
согласования и отправки официальной корреспонденции
