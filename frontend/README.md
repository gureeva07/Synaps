# Email Client Frontend (React + Vite)

Современный почтовый клиент с возможностью генерации автоматических ответов.

## Сборка

```bash
cd frontend
npm install
npm run build
```

Собранные файлы будут в папке `dist/`, которые FastAPI раздаёт автоматически.

## Компоненты

- **Sidebar.jsx** - Боковая панель навигации
- **EmailList.jsx** - Список писем с условным рендерингом
- **EmailViewer.jsx** - Просмотр письма и форма ответа
- **App.jsx** - Главный компонент с интеграцией API

## Особенности

- ✅ Условный рендеринг (письма отображаются только если они есть)
- ✅ Демонстрационные данные для отображения интерфейса
- ✅ Подключение к FastAPI через `api.js`
- ✅ Обработчики кнопок "Отправить" и "Авто-ответ"
